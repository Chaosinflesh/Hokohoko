assessors
=========

A collection of assessment methods that can be used with Hokohoko, which inherit from hokohoko.entities.Assessor. Most notably, includes
the Hokohoko-specific Assessor, **Speculative Accuracy**, but also includes a selection of other common benchmark methods. These are found in the
``hokohoko_assessors`` package.
